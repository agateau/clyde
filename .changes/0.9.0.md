## 0.9.0 - 2026-04-23

### Added

- Added a "comment" entry to the package format. This entry is for package maintenance comments. Unlike regular YAML comments, it is kept when the package file is updated by a tool.

### Fixed

- `clydetools` now includes the pre-release component when extracting a version number from the tag.

- `clydetools check` now supports CalVer version numbers.
